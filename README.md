# CMessenger

This is a simple messenger.

## Tech

- Kotlin
- Jetpack Compose
- Coroutines / Flow
- Gradle Kotlin DSL
- Material 3 (Material YOU)
- Adaptive icons and color scheme
- Firebase Auth, RealtimeDatabase, Firestore, Messaging
- Room Database
- Jetpack navigation
- Hilt (with Dagger 2)
- Coil
- Jetpack Paging library
- Jacoco and Espresso
